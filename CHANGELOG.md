# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and versioning follows SemVer in project practice.

## [0.0.4] - 2026-04-27

### Fixed
- 修复“新建标准表格无法创建”的主流程问题：先提交创建，再分阶段应用格式，避免因某个样式 API 失败导致整批回滚。
- 修复“光标在表格单元格内无法识别当前表格/无法全选”的问题：优先通过 `parentTableOrNullObject` 获取父表格。
- 优化表格格式应用稳定性：边框/布局/文本设置分阶段 `sync`，单阶段失败自动降级，不中断操作。

### Changed
- README 补充“已修复（可用性问题）”说明，并更新版本号到 `0.0.4`。

## [0.0.3] - 2026-04-27

### Added
- 实现“新建标准表格并套格式”功能（支持输入行数、列数）。
- 实现“标准化当前表格”功能（表格对齐、边框、单元格垂直居中、表内字体与段落）。
- 实现“标准化正文段落”功能（两端对齐、首行缩进、段前后 0、1.5 倍行距）。
- 任务窗格 UI 重构为三类核心按钮，直接对应环评报告使用场景。

### Changed
- README 新增“功能实现状态（按需求检查）”说明，并更新版本号到 0.0.3。

## [0.0.2] - 2026-04-24

### Added
- 在 README 中补充“环评报告编制”业务背景与项目目标。
- 新增并细化三类核心需求说明：
  - 自动创建标准表格（含完整表格/字体/段落参数）。
  - 已创建表格一键标准化（同标准参数）。
  - 正文段落一键标准化（两端对齐、首行缩进 2 字符、1.5 倍行距等）。
- 新增 `CHANGELOG.md` 用于记录历史更新。

### Changed
- README 文档结构重组为“已有能力 + 需求整理 + 建议入口 + 版本与变更”，便于后续开发对照实现。

## [0.0.1] - 2026-04-24

### Added
- 初始项目文档：Word 表格格式调整工具说明。
- 基础开发、构建、代码质量命令说明。
