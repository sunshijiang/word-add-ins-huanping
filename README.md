# Word 环评报告格式助手（Office 加载项）

面向环评报告编制场景的 Word 加载项，用于**一键创建和规范化表格/段落格式**，减少重复手工排版工作。

## 项目目标

结合当前项目能力与最新业务诉求，聚焦三个高频动作：

1. **自动创建标准表格并立即套用格式**（指定行列）。
2. **对已存在表格一键应用统一规范格式**。
3. **对正文段落一键应用环评报告常用段落规范**。

---

## 当前能力（已有）

- 提供任务窗格入口，可在 Word 中执行格式化操作。
- 支持多种预设样式（优雅/商务/学术/极简/彩色）。
- 支持手动调整字体、段落、边框等参数。
- 自动检测光标是否位于表格中，并给出状态提示。

---

## 需求整理（本次补充）

> 以下为环评报告编制的目标“标准格式模板”，应在加载项中以按钮/快捷操作方式快速应用。

### 1）自动创建标准表格（新建表格 + 套格式）

- 动作：创建指定行列的表格（由用户输入行数、列数）。
- 表格属性：
  - 尺寸：指定宽度 100%
  - 对齐方式：居中
  - 文字环绕：无
  - 边框和底纹：
    - 上下边框：1.5 磅
    - 左右边框：无
    - 内部边框：0.5 磅
  - 行：无指定高度
  - 列：无指定宽度
  - 单元格：垂直对齐方式居中
- 字体：
  - 中文字体：宋体，五号
  - 西文字体：Times New Roman，五号
- 段落：
  - 对齐方式：居中
  - 缩进：文本前后缩进 0
  - 特殊缩进：无
  - 间距：段前段后 0
  - 行距：单倍行距

### 2）已创建表格的标准化设置（仅套格式）

- 动作：对当前选中表格（或光标所在表格）应用统一格式。
- 格式规则与“自动创建标准表格”一致：
  - 表格属性：宽度 100%、居中、无环绕、上下 1.5 磅、左右无、内部 0.5 磅、无固定行高/列宽、单元格垂直居中。
  - 字体：中文宋体五号；西文 Times New Roman 五号。
  - 段落：居中、前后缩进 0、无特殊缩进、段前后 0、单倍行距。

### 3）正文段落标准化设置

- 动作：对当前选中段落（或当前段落）应用正文格式。
- 段落规则：
  - 对齐方式：两端对齐
  - 缩进：文本前后缩进 0
  - 特殊缩进：首行缩进 2 字符
  - 间距：段前段后 0
  - 行距：1.5 倍行距

## 建议的功能入口与快捷操作

- 任务窗格增加三个主按钮：
  - `新建标准表格`
  - `标准化当前表格`

## 功能实现状态（按你的需求检查）

- ✅ **自动创建标准表格**：已支持输入行列并一键创建，同时应用标准表格/字体/段落格式。
- ✅ **已建表格标准化**：已支持对光标所在表格一键套用标准格式。
- ✅ **正文段落标准化**：已支持对选中段落应用两端对齐、首行缩进、1.5 倍行距等规则。
- ⚠️ **兼容性说明**：表格宽度 100% 与边框粗细依赖 Word API 版本，已实现优先调用；在较旧版本下可能出现部分属性降级。

---

## 已修复（可用性问题）

- 修复“新建标准表格无结果”的问题：创建动作与格式动作拆分提交，确保即使个别样式设置失败，表格也能成功创建。
- 修复“光标在单元格内无法识别/无法全选表格”的问题：优先使用 `parentTableOrNullObject` 检测当前父表格，并在按钮中强制二次检测。
- 增加格式应用降级机制：边框和部分表格属性按阶段提交，失败自动降级而不中断主流程。
